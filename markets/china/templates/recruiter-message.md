# 招聘者 / 猎头私信模板

## 结构

1. 简短说明来意。
2. 用 2-3 个事实说明匹配点。
3. 说明目标岗位、城市、到岗时间或薪资边界。
4. 请求下一步沟通。

## 模板

```text
您好，我看到您发布的【公司/岗位】机会，想进一步了解。

我的背景和岗位有几处比较相关：
1. 【证据支持的匹配点 1】
2. 【证据支持的匹配点 2】
3. 【证据支持的匹配点 3】

我目前重点看【目标岗位/方向】，城市偏好是【城市/远程/混合】，到岗时间大约【到岗时间】。如果岗位核心要求和我的背景匹配，方便的话想约一个时间进一步沟通。
```

## 猎头版本

```text
您好，我目前在看【目标方向】机会，偏好【城市/行业/公司阶段】。我的核心背景是【一句话定位】，有【证据支持的经历】。

如果您手上有【目标岗位】相关机会，欢迎发我 JD。我会先看职责、团队方向、薪资范围和工作地点，再判断是否合适推进。
```

## 薪资边界版本

```text
您好，我对这个岗位感兴趣。为了节省双方时间，想先确认下薪资范围是否大致在【期望范围】附近，以及工作地点/办公方式是否为【地点/办公方式】。如果匹配，我可以发详细简历并进一步沟通。
```
