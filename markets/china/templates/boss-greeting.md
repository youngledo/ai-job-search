# BOSS 直聘打招呼模板

使用原则：

- 控制在 1-2 句话。
- 先说匹配点，再表达希望进一步沟通。
- 不写无法证明的年限、规模、业绩或大厂背景。
- 如果岗位匹配度一般，坦诚使用“相关经验”或“相近项目经验”。

## 直接匹配

```text
您好，我关注到这个岗位主要需要【核心技能/方向】。我有【相关经历/项目】经验，和岗位中的【关键要求】比较匹配，想进一步了解团队和职责。
```

短版：

```text
您好，我有【核心技能/方向】相关经验，和这个岗位的【关键要求】比较匹配，想进一步沟通一下。
```

## 相邻背景 / 转方向

```text
您好，我目前的经历主要在【已有方向】，其中【可迁移能力/项目】和岗位要求的【目标方向】有交集。想了解这个岗位对【关键能力】的侧重点。
```

## 有经验候选人

```text
您好，我在【领域/岗位】方向有【可证明经验】经验，做过【项目/职责】。看到岗位关注【关键要求】，这部分和我的经历比较贴合，想进一步沟通。
```

## 应届 / 早期候选人

```text
您好，我是【学校/专业/背景】背景，做过【项目/实习/课程项目】。岗位中的【技能/方向】和我的项目经历比较相关，想了解是否有机会沟通。
```

## 谨慎询问信息

```text
您好，想确认下这个岗位的主要工作内容更偏【方向 A】还是【方向 B】？我有【相关经历】，想判断匹配度后再详细沟通。
```
