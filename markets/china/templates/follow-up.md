# 跟进消息模板

## 初次沟通后未回复

```text
您好，打扰了。我之前咨询过【岗位名称】岗位，想补充一下我的匹配点：【一句话证据支持匹配点】。如果岗位还在招聘，方便的话想进一步了解团队和职责。
```

## 约面确认

```text
您好，已收到面试邀请。我这边【可面试时间 1】或【可面试时间 2】方便。请问面试形式、预计时长和主要考察方向是什么？我好提前准备。
```

## 面试后感谢

```text
您好，感谢今天的面试沟通。通过交流我对【岗位/团队方向】有了更清晰的了解，也觉得自己在【匹配点】上可以比较快产生贡献。后续如需补充材料，我可以继续提供。
```

## 面试后状态询问

```text
您好，想跟进一下【岗位名称】的面试进展。我对这个机会仍然感兴趣，也愿意补充项目材料或进一步沟通。请问目前流程大概到哪个阶段了？
```

## Offer 信息确认

```text
您好，感谢 offer。为了完整评估，我想确认几个信息：薪资结构、试用期规则、绩效/年终机制、工作地点与办公方式、团队汇报关系、入职后的核心目标。确认后我会尽快反馈决定。
```

## 礼貌拒绝

```text
您好，感谢这次机会和沟通。综合岗位方向、个人规划和当前选择，我这次先不继续推进。感谢您的时间，也祝招聘顺利。
```
