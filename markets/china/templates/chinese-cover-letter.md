# 中文求职信 / 邮件模板

## 使用原则

- 面向正式邮件、官网投递补充说明或内推材料。
- 控制在 4-6 段。
- 每个能力点都要有证据，不写空泛热情。
- 如果是 BOSS 直聘聊天，不要直接使用全文，改用 `boss-greeting.md`。

## 模板

```text
您好，

我想申请【公司】的【岗位名称】岗位。我关注到这个岗位重点需要【岗位核心要求 1】、【岗位核心要求 2】和【岗位核心要求 3】。结合我的经历，我认为自己在【最强匹配方向】上与岗位有较高匹配度。

在【经历/项目 1】中，我负责【你的动作】，使用/涉及【技能或方法】，最终取得【可证明结果】。这段经历对应岗位中的【岗位要求】。

在【经历/项目 2】中，我主要解决【问题】，通过【方案】推进【结果】。这让我积累了【业务/协作/技术能力】，也能支持岗位中【职责】部分的工作。

我目前希望寻找【目标方向】相关机会，偏好【城市/办公方式/行业】。如果有机会加入，我希望优先在【可贡献方向】上产生价值，同时继续补强【诚实的成长方向】。

感谢您阅读我的申请。期待有机会进一步沟通。

此致
敬礼

【姓名】
【电话 / 邮箱 / 微信】
```

## 精简邮件版

```text
您好，我想申请【岗位名称】岗位。

我的背景主要是【一句话定位】。我看到岗位重点关注【关键要求】，这与我在【项目/工作】中的【证据支持经历】比较匹配。附件是我的简历，里面包含更完整的项目和经历说明。

如果岗位仍在招聘，期待有机会进一步沟通。

谢谢！
【姓名】
```
